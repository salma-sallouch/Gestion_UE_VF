# 🎓 Gestion UE - API REST

Projet universitaire permettant de gérer les **Unités d’Enseignement (UE)** et les **Modules**.

## 📌 Technologies utilisées
- Java
- JAX-RS (API REST)
- Apache Tomcat
- HTML / CSS / JavaScript
- IntelliJ IDEA

## ⚙️ Fonctionnalités

### Gestion des UE
- 📄 Lister les UE
- ➕ Ajouter une UE
- ✏️ Modifier une UE
- 🗑 Supprimer une UE

### Gestion des Modules
- ➕ Ajouter un module
- 🗑 Supprimer un module

## 🚀 Lancer le projet
1. Lancer **Tomcat**
2. Ouvrir le navigateur
3. Accéder à :
http://localhost:8089/Gestion_UE_VF
## 📷 Interface

Interface web permettant d’interagir avec l’API REST.
